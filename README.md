Tech Stack

- Frontend: HTML, CSS, JavaScript (Vite + TailwindCSS)  
- Backend: Node.js + Express.js  
- Database: MongoDB / Firebase  
- Storage: Cloudinary

How to Run Locally

1. Install dependencies
